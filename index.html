Setting up Playground

Om te playground te kunnen zien en er effectief mee te werken zal je enkele stappen moeten volgen om deze te openen:

Open Github Codespace
In de Kallistar Github pagina zal je een repository zien met de naam "tailwind-html-example". Als je op de repo klikt, zal je in de rechterbovenhoek een groene knop zien met de naam "Code"
Click Codespaces
Wanneer je op de groene knop klikt, zal je twee tabs zien. De tweede tab heeft de naam "Codespace". Als je op deze tab klikt zou je normaal gezien de repo in de Codespace Terminal kunnen openen.
Run the Code
Je kan de pagina openen door in de Terminal "npm run dev" te typen, wat op zijn beurt een link zal tonen. Open deze link. In de HTML file kan je alle sourcecode vinden voor deze pagina.

HTML & CSS Code

Een van de gemakkelijkheden bij het gebruik van Tailwind is dat de CSS is in de HTML code. Als je componenten gebruikt kan je deze in een aparte library stoppen en ophalen. Dan kan je de componenten aanpassen op globaal vlak of lokaal in de html code. Bovendien heeft Tailwind uitgebreide documentatie dat stap voor stap uitlegt hoe alles werkt, documentatie rond alle classes, en zoveel meer.

Ik zou op zijn beurt, met behulp van Tailwind, een geperzonaliseerde library maken dei aansluit bij jou bedrijfsidentiteit. Op deze manier zal je alle middelen hebben om eenvoudig aanpassingen te kunnen maken.

Het doel ban deze library is om alles vlotter te laten verlopen en vlotter aanpassingen te kunnen maken door de inginieurs, zonder tussenkomst van een externa partij.

Heb je nog vragen in verband met de library, het toepassen, of andere vragen, geef me gerust een seintje. Voel je vrij om de playgound uit te testen en om je wat vertrouwder te maken met de werking.
